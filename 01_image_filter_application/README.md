# Image Filter Application

## Overview
A command-line Python application that applies three fundamental image processing filters to any input image.

## Features
- **Grayscale Conversion**: Converts color images to black and white
- **Gaussian Blur**: Applies smoothing with a 7x7 kernel
- **Canny Edge Detection**: Highlights structural edges in the image
